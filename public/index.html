UptimeRobot Setup

Keep your automation running 24/7

Setup Guide
1

Deploy Backend to Railway/Render

Push the backend/ folder to GitHub, then connect it to Railway or Render. Get a public URL like https://wareach.up.railway.app

2

Register on UptimeRobot

Create a free account at uptimerobot.com. Free plan supports 50 monitors with 5-minute checks.

3

Add HTTP Monitor

Click "Add New Monitor" → Type: HTTP(s) → URL: https://your-backend.railway.app/ping → Interval: 5 minutes → Save.

4

Keep WhatsApp Web Open

The Tampermonkey script runs in your browser. Keep a dedicated browser window open on web.whatsapp.com on a PC that stays on 24/7 or use a VPS with a remote browser like Browserless.

5

Keepalive Pings

The script sends a GET /ping request every 4 minutes to keep your Railway/Render free tier awake. UptimeRobot does the same from outside — double protection.
